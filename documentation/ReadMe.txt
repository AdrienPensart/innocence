Attention ce ceci est un programme dangeureux, qu'il faut utiliser uniquement
sur un réseau que l'utilisateur possède.

Le trojan (Innocence) se compose de deux exécutables : le client (Inhibitor) et le serveur (TheSleeper).
Contrairement aux autres trojans, le client est le programme à installer sur la machine de la victime
et le serveur est le programme qui va sur le machine du pirate.
Ce changement de dénomination est en rapport avec le fait qu'un pirate peut contrôler plusieurs machines victimes,
ce sont les clients, d'autre part, ce sont les machines victimes qui se connectent à la machine du pirate.

Le serveur contient en plus de la partie de contrôle des victimes, un module pour générer des exécutables clients avec des
paramètre différents (port, mot de passe d'accès à la machine infectée, nom de la victime, etc)
